# Palavu Centre - Restaurant Website

Modern, mobile-first restaurant website for Palavu Centre, featuring authentic Godavari heritage cuisine.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📋 Features

- ✅ Mobile-first responsive design
- ✅ Dark theme with turmeric gold accents
- ✅ Full menu with categories and search
- ✅ Shopping cart functionality
- ✅ Gallery with lightbox
- ✅ Customer reviews section
- ✅ Contact form with Google Maps
- ✅ Catering packages
- ✅ Franchise inquiry form
- ✅ WhatsApp floating button
- ✅ Social media integration

## 🎨 Tech Stack

- React 18
- Vite
- Tailwind CSS
- Google Fonts (Playfair Display + Inter)
